# fiction-titles Указатель предназначен для определения имени автора по заглавию художественного произведения.
Представлены художественные произведения, изданные в виде отдельных книг, на русском языке и в переводах с иностранных языков и языков народов СССР с 1801 по 1985 год. Включены издания «вольной печати» и зарубежных издательств, сотрудничавших с РСФСР в 1920-е гг.
Содержит около 130000 названий книг, сборников, рассказов, стихотворений, фельетонов, пьес.
Подробнее о указателе: http://nlr.ru/res/inv/guideseria/liter/record_full.php?record_ID=126687&rid=1673
